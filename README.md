# Real-Time Data Analytics Platform

## Problem Statement
Build an end-to-end data pipeline that ingests user activity data, processes it using Apache Spark, stores it efficiently, and generates analytical insights.

## Architecture Overview
- Python-based data generator
- Apache Spark (PySpark) for data processing
- MongoDB for raw data storage
- MySQL for processed analytics data
- Power BI for dashboards
- Azure Databricks for cloud execution
